# 외부 레퍼런스 (References)

이 디렉토리는 DrinkSomeWater 앱 개발 중 참고한 외부 레퍼런스를 관리합니다.

## 주요 참고 자료 카테고리

### Apple 공식 문서
- HealthKit 프레임워크
- WidgetKit 가이드
- SwiftData 공식 문서
- SwiftUI 레퍼런스
- Core Data 아키텍처

### WWDC 세션
- 2023-2024 WWDC 세션 링크
- iOS 개발 관련 주요 발표 자료

### Human Interface Guidelines (HIG)
- iOS 디자인 가이드라인
- 접근성 (Accessibility) 기준

### Swift API Design Guidelines
- Swift.org 공식 API 가이드
- 코딩 스타일 및 네이밍 컨벤션

### 디자인 패턴 및 아키텍처
- MVVM, Clean Architecture 관련 자료
- Reactive Programming 패턴

## 레퍼런스 추가 방법

새로운 레퍼런스를 추가할 때는 다음 규칙을 따릅니다:

1. 파일명: `[카테고리]_[주제].md` (예: `healthkit_water-tracking.md`)
2. 각 파일에는 출처 URL과 간단한 요약 포함
3. 시간순이 아닌 카테고리별로 관리
4. 더 이상 유효하지 않은 링크는 주기적으로 검토

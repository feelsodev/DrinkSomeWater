## How to Add a Reference

Follow these rules when adding a new reference:

1. File name: `[category]_[topic].md` (e.g., `healthkit_water-tracking.md`)
2. Each file should include the source URL and a brief summary
3. Organize by category, not chronologically
4. Periodically review and remove links that are no longer valid

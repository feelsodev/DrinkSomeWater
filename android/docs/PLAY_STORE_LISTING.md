## 그래픽 자산

### 앱 아이콘
- 크기: 512 x 512 px
- 형식: PNG (32-bit, 알파 포함)
- 파일명: `ic_launcher.png`

### 특성 그래픽 (Feature Graphic)
- 크기: 1024 x 500 px
- 형식: JPEG 또는 24-bit PNG
- 파일명: `feature_graphic.png`

### 스크린샷 (필수)

#### 스마트폰 (최소 2장, 최대 8장)
- 권장 비율: 16:9 또는 9:16
- 최소 크기: 320 px
- 최대 크기: 3840 px

필요한 스크린샷:
1. 홈 화면 - 물결 애니메이션과 퀵버튼
2. 홈 화면 - 목표 달성 상태
3. 기록 화면 - 캘린더 뷰
4. 기록 화면 - 타임라인 뷰
5. 설정 화면
6. 위젯 (3가지 크기)
7. 온보딩 화면

#### Wear OS (선택, 최대 8장)
- 원형 스크린샷 권장

필요한 스크린샷:
1. 워치 홈 화면
2. 퀵 추가 화면
3. 직접 입력 화면
